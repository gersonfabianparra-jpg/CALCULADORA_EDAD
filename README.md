# Calculadora de Edad y Gestión de Personas

Aplicación escrita en **Python** con **Tkinter** que permite:

- Ingresar nombre, dirección y fecha de nacimiento.
- Calcular la edad en años.
- Mostrar el día de la semana en que nació la persona.
- Mostrar cuántos días faltan para el próximo cumpleaños.
- Registrar varias personas en una tabla en memoria, lista para ser conectada a una base de datos en el futuro.

Este proyecto se utiliza como demostración de:
- Lógica de negocio separada de la interfaz gráfica.
- Uso de Tkinter y ttk para una interfaz profesional.
- Preparación para persistencia con base de datos (por implementar).
